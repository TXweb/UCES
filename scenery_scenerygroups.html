UCES presents 6 never-before-seen Scenario Groups. These objects have been collected into their own tabs which are selectable from the RCT2 Object Editor's "Object Selection - Themes" window. If you don't want to use all of the items from the tab, and save tab space for other theme collections you can select the objects from the other selection windows and they will show up in the "?" theme tab.
